# 编钟游戏光标设置指南

## 光标素材准备

为了实现编钟游戏的特殊光标效果，您需要准备以下两种光标贴图：

1. **默认光标贴图（defaultCursor）**：
   - 尺寸建议：32x32 或 64x64 像素
   - 格式：PNG格式，支持透明通道
   - 内容：可以设计为木槌/鼓槌的静止状态

2. **点击光标贴图（clickCursor）**：
   - 尺寸建议：与默认光标贴图相同
   - 格式：PNG格式，支持透明通道
   - 内容：设计为木槌/鼓槌的击打状态

## 光标设置步骤

1. 在Unity编辑器中，选择包含BellGameMainMenu组件的游戏对象
2. 在Inspector面板中查找CursorManager组件
3. 将准备好的光标贴图拖拽到对应的字段中：
   - 默认光标贴图 → defaultCursor
   - 点击光标贴图 → clickCursor
4. 设置光标热点位置（cursorHotspot）：
   - 热点是光标的精确点击位置，通常设为图像上槌头的位置
   - X坐标：从左到右，0表示最左边
   - Y坐标：从上到下，0表示最上边
5. 设置点击动画持续时间（clickAnimationDuration）：
   - 建议值：0.1-0.2秒，时间太长会影响操作流畅度

## 实现效果

- 进入游戏时，光标自动变为默认木槌光标
- 在游戏的所有界面（主菜单、游戏界面、自由演奏、背景介绍等）都保持自定义光标
- 玩家点击鼠标左键时，光标暂时变为点击状态的木槌，模拟敲击动作
- 点击动画结束后自动恢复默认木槌光标
- 只有在玩家通过退出按钮完全退出游戏时，光标才会恢复为系统默认光标

## 光标状态管理

游戏内的光标状态流转如下：
1. 进入游戏 → 设置为自定义默认光标
2. 点击鼠标 → 短暂切换为点击光标 → 自动恢复为默认光标
3. 返回主菜单 → 保持自定义默认光标不变
4. 退出游戏 → 恢复为系统默认光标

## 光标图像设计建议

- 使用明亮的颜色，确保在游戏界面上容易识别
- 木槌部分应当清晰可辨，与编钟主题相符
- 点击状态应当明显区别于默认状态，以提供良好的视觉反馈
- 考虑为光标添加简单的阴影或边缘，增强视觉效果
- 保持简洁设计，避免过于复杂的细节影响识别
